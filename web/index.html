문서 반영 방식

새 Markdown은 요청으로 접수되고 관리자 승인 후 검색 문서에 반영됩니다. 기존 문서는 같은 doc_key로 갱신 요청을 보냅니다.

문서 삭제/비활성화는 현재 기본 MVP 화면에서 제공하지 않습니다.

기본 경로에서는 자동 파일명과 컬렉션 기본값을 사용합니다.
비워두면 `upload_타임스탬프.md` 형식으로 자동 생성됩니다.
기존 문서 갱신 시 같은 `doc_key`를 사용합니다.
